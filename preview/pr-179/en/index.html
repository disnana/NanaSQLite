NanaSQLiteHigh-Performance SQLite Dict Wrapper with Caching

Use SQLite like a Dictionary.
Fast, Safe, Thread-Safe Python Library.

Get Started
GitHub

Fast Memory Caching

Choose from UNBOUNDED, LRU, and TTL cache strategies. Supports lazy loading, bulk loading, and optional lru-dict C-extension acceleration.

Security & Encryption

Combines strict SQL validation with transparent AES-GCM/ChaCha20/Fernet encryption (v1.3+) to fully protect your data.

Async Support

Delivers maximum performance in asynchronous environments like FastAPI with a custom thread pool.

Intuitive API

Just like a Python dictionary. Persistence without boilerplate code.

V2 Non-blocking Engine

Optional dual-lane write-back architecture with staging buffer, priority queue, and Dead Letter Queue for write-heavy workloads.

Full Documentation

Bilingual (EN/JA) documentation covering cache strategies, encryption, V2 architecture, exceptions, and security audit.
