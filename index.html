NanaSQLite高性能キャッシュ搭載 SQLiteベース・ディクトラッパー

SQLiteを辞書感覚で。
高速、安全、スレッドセーフなPythonライブラリ。

GitHub

高速メモリキャッシュ

UNBOUNDED・LRU・TTL の 3 つのキャッシュ戦略に対応。遅延ロード・一括ロード、lru-dict C 拡張による高速化もサポート。

セキュリティと暗号化

厳格なSQL検証に加え、AES-GCM/ChaCha20/Fernetによる透過的暗号化 (v1.3+) を標準装備。機密データを強力に保護します。

非同期 (Async) 対応

独自のスレッドプールにより、FastAPI等の非同期開発で最高のパフォーマンスを発揮します。

直感的なAPI

Pythonの辞書そのもの。ボイラープレートなしで永続化を実現します。

V2 非ブロッキングエンジン

デュアルレーン書き込みバックアーキテクチャ。ステージングバッファ・優先度キュー・Dead Letter Queue を備えた書き込み負荷の高いワークロードに最適。

充実したドキュメント

日本語・英語のバイリンガルドキュメント。キャッシュ戦略・暗号化・V2 アーキテクチャ・例外クラス・セキュリティ監査を網羅。
